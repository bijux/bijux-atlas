# atlasctl

Internal Python control-plane package for `bijux-atlas`.

## Install (editable)

```bash
python -m pip install -e packages/atlasctl
```

## Quickstart

```bash
python -m atlasctl --help
python -m atlasctl suite run fast --json
```
