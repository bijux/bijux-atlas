usage: atlasctl [global options] <group> ...

control-plane groups:
  docs       documentation and docs contracts
  configs    configuration validation and sync
  dev        development checks, suites, and tests
  ops        operations and runtime orchestration
  policies   policy enforcement and culprits
  internal   internal/legacy compatibility and diagnostics

run `atlasctl <group> --help` for group commands.

options:
  -h, --help            show this help message and exit
  --version             show program's version number and exit
  --json                emit JSON output
  --run-id RUN_ID       run identifier for artifacts
  --dry-run             resolve and report command without mutating side
                        effects
  --artifacts-dir ARTIFACTS_DIR
                        artifacts/evidence root path
  --evidence-root EVIDENCE_ROOT
                        evidence root path
  --run-dir RUN_DIR     override run directory root
  --cwd CWD             run command from an explicit repository root
  --profile PROFILE     profile id
  --format {text,json}  output format
  --log-json            emit structured JSON logs to stderr
  --network {allow,forbid}
                        network access mode (default: forbid)
  --allow-network       explicitly allow network when command group policy
                        permits it
  --no-network          deprecated alias for --network=forbid
  --require-clean-git   fail if git workspace is dirty
  --verbose             enable verbose diagnostics
  --quiet               only emit errors
