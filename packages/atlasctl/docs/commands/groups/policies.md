# Policies Command Group

Commands mapped to `policies` effects policy.

## Commands

- `policies`

## Examples

- `atlasctl policies explain budgets`
- `atlasctl policies check-dir-entry-budgets --print-culprits --top 20`
- `atlasctl policies check-py-files-per-dir --json`
- `atlasctl report budgets --by-domain --json`
