# Ops Command Group

Commands mapped to `ops` effects policy.

## Commands

- `k8s`
- `load`
- `obs`
- `ops`
- `stack`

## Examples

- `atlasctl k8s conformance --report json`
- `atlasctl load run --report json`
- `atlasctl obs checks --report json`
- `atlasctl ops status --report json`
- `atlasctl stack status --report json`
