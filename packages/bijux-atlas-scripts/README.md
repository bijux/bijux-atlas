# bijux-atlas-scripts (packages/)

Package scaffold for the internal tooling CLI.

Toolchain SSOT: `pip-tools`-style lockfiles.
CLI framework: `Typer`.

Examples:

```bash
python -m bijux_atlas_scripts --help
python -m bijux_atlas_scripts --version
atlas-scripts doctor --json
atlas-scripts gates list
atlas-scripts gates run scripts-check
bijux-atlas-scripts doctor --json
```

This package is internal-only and follows repository licensing policy in the root `LICENSE`.
