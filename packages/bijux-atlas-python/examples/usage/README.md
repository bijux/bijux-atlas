# bijux-atlas-usage-examples

Repository-style folder with Python SDK usage snippets.

- `basic/simple_query.py`
- `advanced/streaming_pipeline.py`

## Run locally

Install the package in editable mode:

```bash
python -m pip install -e packages/bijux-atlas-python
```

Set runtime endpoint for examples:

```bash
export BIJUX_ATLAS_URL=http://127.0.0.1:8080
```

Run an example:

```bash
python packages/bijux-atlas-python/examples/usage/basic/simple_query.py
```
