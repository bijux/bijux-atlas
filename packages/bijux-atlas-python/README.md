# bijux-atlas (Python SDK)

`bijux-atlas` is the Python SDK package for Bijux Atlas.

## Install

```bash
pip install bijux-atlas
```

## Import

```python
from bijux_atlas import BijuxAtlasClient
```

## Packaging scope

- SDK source: `src/bijux_atlas/`
- Examples: `examples/`
- Tests: `tests/`
- Docs: `docs/`
- Notebooks: `notebooks/`
