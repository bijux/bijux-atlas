# Config examples

This directory is the only allowed location for example config files.

Rules:
- examples must be deterministic text files
- examples must map to a declared schema when machine-readable
- examples must be referenced from `configs/README.md`
- examples must explain what they demonstrate and what they do not prove

Runtime examples:
- `runtime/server-minimal.toml`
- `runtime/server-observability.toml`

## Dataset examples

- `datasets/atlas-example-minimal`
- `datasets/atlas-example-medium`
- `datasets/atlas-example-large-synthetic`
- `datasets/specification.md`

See `runtime/README.md` and `datasets/README.md` for the intent and limits of each example set.
