# Runtime example configs

These files are illustrative runtime inputs, not the authoritative runtime policy surface.

Use them to:
- boot a minimal local server quickly
- compare a minimal setup with an observability-enabled setup
- test documentation and tutorial flows without editing governed production inputs

Do not use them to infer:
- production readiness
- operator policy
- release support guarantees

Authoritative runtime inputs live under `configs/sources/runtime/`.
