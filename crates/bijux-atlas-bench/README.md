# bijux-atlas-bench

Benchmark models and harness primitives for reproducible Atlas performance evidence.

## Purpose

Provide stable benchmark data structures and validation rules shared by Atlas performance tooling.

## How to use

Import this crate from benchmark runners and report generators to load benchmark config, dataset tier metadata, and normalized result records.

## Where docs live

- Crate docs index: `crates/bijux-atlas-bench/docs/index.md`
- Contract: `crates/bijux-atlas-bench/CONTRACT.md`

## Modules

- `config`: benchmark environment and reproducibility configuration.
- `dataset`: benchmark dataset tiering and registry metadata.
- `harness`: benchmark result structure and validation.
