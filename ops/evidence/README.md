# Ops Evidence Bundle

This directory defines the versioned schema for the operations evidence bundle.

- Schema root: `ops/evidence/schema/v1/ops-evidence-bundle.schema.json`
- Producer: `bijux dev atlas ops evidence collect`
- Verifier: `bijux dev atlas ops evidence verify`
- Comparator: `bijux dev atlas ops evidence diff`

The bundle is designed to be verifiable offline and deterministic across identical inputs.
