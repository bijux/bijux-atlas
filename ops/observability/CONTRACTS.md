# Observability Contracts Pointer

Canonical observability contracts live under `ops/observability/contract/`.

- metrics: `ops/observability/contract/metrics-contract.json`
- alerts: `ops/observability/contract/alerts-contract.json`
- dashboard panels: `ops/observability/contract/dashboard-panels-contract.json`
- logs fields: `ops/observability/contract/logs-fields-contract.json`

Do not add duplicate contract JSON files at `ops/observability/` root.
