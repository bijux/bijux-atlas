# Maintainership transition guide

## Transition checklist

- Confirm outgoing and incoming maintainer overlap window
- Transfer domain ownership registry entries
- Hand over operational runbooks and incident context
- Revalidate governance and security command baselines
- Publish transition evidence in governance report bundle
