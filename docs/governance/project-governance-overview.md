# Project governance overview

Atlas governance combines policy documents, registry-backed controls, CLI validation, and auditable evidence artifacts.

Primary controls:

- Rule registry in `configs/governance/enforcement/rules.json`
- Validation commands under `bijux-dev-atlas governance ...`
- Evidence reports under `artifacts/governance/` and `ops/governance/`
