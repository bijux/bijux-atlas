# Maintainers and Roles

## Purpose

This page names the durable governance roles used by the repository and the
areas they are expected to maintain.

## Roles

- `atlas-governance`: repository governance policy, review expectations, and
  registry integrity
- `atlas-docs`: canonical documentation, reader navigation, and public docs
  accuracy
- `bijux-atlas-operations`: operations inventories, pins, tools, and release
  evidence surfaces

## Working Agreement

Each role is responsible for keeping its authoritative files reviewable,
validated, and synchronized with generated evidence.

## Stability

These roles are stable governance labels. Update this page when ownership names
or their maintained scopes change.
