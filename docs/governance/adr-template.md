# ADR Template

## Purpose

This page provides the minimum structure expected for a durable decision record.

## Template

```md
# ADR: <clear decision title>

## Context

## Decision

## Consequences

## Source-of-Truth Files
```

## Stability

This template is stable guidance. Adjust it only when the repository changes
the required shape of decision records.
