# Maintainership policy

## Maintainer responsibilities

- Review and approve changes in owned domains
- Enforce release, compatibility, and security policies
- Keep operational and governance documentation current
- Participate in incident response and post-incident action tracking

## Eligibility

- Consistent review quality over multiple release cycles
- Demonstrated ownership of domain reliability and docs
- Adherence to governance and security controls

## Rotation

- Roles are reviewed quarterly.
- Coverage gaps require temporary backup assignment.
