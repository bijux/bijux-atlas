# ADR-0006: Tutorial structure and consolidation

- Status: accepted
- Date: 2026-03-05
- Owners: @atlas-governance

## Context

Tutorial proliferation without consolidation creates duplicated paths and conflicting guidance.

## Decision

Tutorial navigation is curated into a smaller canonical learning sequence with generated references and explicit consolidation policy.

## Consequences

1. User onboarding path is shorter and clearer.
2. Duplicate tutorial drift is reduced through governance checks.
