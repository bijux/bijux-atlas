# Deprecation lifecycle

1. Introduce replacement and announce deprecation.
2. Record deprecation metadata in `configs/governance/deprecations.yaml`.
3. Keep redirects/migrations active during supported window.
4. Remove deprecated surface only after removal target date and evidence update.
