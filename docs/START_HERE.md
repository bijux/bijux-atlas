# Start Here

Use one command path to validate local developer flow:

```bash
bijux dev atlas demo quickstart --format json
```

The quickstart contract is a deterministic four-step plan:

1. Bring up local stack profile.
2. Ingest the tiny fixture set.
3. Run one query smoke call.
4. Read one metrics endpoint.

Budgets:

- Maximum steps: `4`
- Maximum duration budget: `3` minutes

Onboarding entrypoint policy:

- This page is the only onboarding root.
- Quickstart pages must remain at `<= 5` documents under `docs/quickstart/`.
