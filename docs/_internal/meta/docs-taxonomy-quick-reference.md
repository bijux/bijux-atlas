# Docs taxonomy quick reference

## Audiences

- `user`
- `operator`
- `contributor`

## Page types

- `concept`
- `guide`
- `runbook`
- `reference`
- `policy`
- `adr`

## Stability

- `draft`
- `stable`
- `deprecated`

## Rules

- Stable pages require ownership metadata.
- Procedures require verification guidance.
- Deprecated pages require a replacement link.
