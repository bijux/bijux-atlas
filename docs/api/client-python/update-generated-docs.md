# Update generated client docs

Client docs are generated by `bijux-dev-atlas`, not local scripts.

## Regenerate

```bash
cargo run -p bijux-dev-atlas -- clients docs-generate --client atlas-client --format json
```

## Verify

```bash
cargo run -p bijux-dev-atlas -- clients docs-verify --client atlas-client --format json
```

Generation inputs:

- `configs/clients/atlas-client-docs.json`
- `configs/openapi/v1/openapi.snapshot.json`
