# Stability Legend

Use these values in documentation front matter:

- `stable`: compatibility is expected for normal upgrades.
- `experimental`: surface can change without long deprecation windows.
- `deprecated`: surface remains available short term and has a replacement path.
- `internal`: page is for maintainers and not part of public promises.
