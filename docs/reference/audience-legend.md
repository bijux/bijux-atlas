# Audience Legend

Use these values in documentation front matter:

- `user`: consumer of Atlas APIs and dataset surfaces.
- `operator`: service owner running Atlas deployments.
- `contributor`: engineer changing code, contracts, or operations assets.
- `internal`: maintainer-only governance and generated evidence pages.
