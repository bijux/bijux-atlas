# Ops artifacts reference

## Canonical ops surfaces

- `ops/charts`: Helm chart packaging inputs.
- `ops/profiles`: values profiles and deployment presets.
- `ops/inventory`: governed inventory and coverage manifests.
- `ops/schemas`: operational schema contracts.
- `ops/load`: load and performance scenario definitions.
- `ops/reports`: generated operational evidence outputs.
