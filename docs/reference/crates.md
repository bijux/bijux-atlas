# Crates reference

## Workspace crates

- `bijux-atlas-api`: shared API models and contract surface.
- `bijux-atlas-cli`: end-user runtime CLI.
- `bijux-atlas-core`: core domain primitives and utilities.
- `bijux-atlas-ingest`: ingestion pipeline primitives.
- `bijux-atlas-model`: data model and schema-level semantics.
- `bijux-atlas-policies`: policy definitions and validation.
- `bijux-atlas-query`: query language and execution helpers.
- `bijux-atlas-server`: service runtime and endpoint surface.
- `bijux-atlas-store`: storage abstractions.
- `bijux-dev-atlas`: control-plane and governance tooling.
