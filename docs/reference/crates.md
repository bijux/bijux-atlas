# Crates reference

## Workspace crates

- `bijux-atlas`: end-user runtime crate with embedded API, ingest, store, client, server, model, query, and policy modules.
- `bijux-atlas-python`: private Python SDK distribution crate with package metadata, compatibility payloads, and optional native bindings.
- `bijux-dev-atlas`: private control-plane crate for repository governance, docs/config validation, release workflows, ops orchestration, and benchmark automation.
