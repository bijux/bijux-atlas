# Production-ready Boundaries

## In scope

- Governed deploy and release workflows.
- Contract-validated API and operational surfaces.
- Audit-oriented evidence artifacts and diagnostics.

## Out of scope

- Unvalidated ad-hoc deployment paths.
- Ungoverned schema and policy bypasses.
- Manual release actions without generated evidence.
