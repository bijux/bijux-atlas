# Repository Structure

- Owner: `docs-governance`
- Stability: `stable`

Top-level structure:
- `crates/`: Rust workspace packages
- `configs/`: schemas, policies, and contract data
- `ops/`: operational contract files and environment manifests
- `docs/`: canonical documentation source
- `makefiles/`: stable make command implementations
- `docker/`: container and image build contracts
