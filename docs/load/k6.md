# Atlas Load Harness (k6)

Deterministic load scenario for Phase 11 observability.

## Run

```bash
ATLAS_BASE=http://127.0.0.1:3000 k6 run load/k6/atlas_phase11.js
```

## Reproducibility

- Fixed scenario (`constant-vus`, fixed duration)
- Fixed endpoints and query parameters
- Threshold assertions baked into script
