# Tutorial: Analyzing Evidence Bundles

## Goal

Inspect generated evidence bundles and validate release or ops claims.

## Steps

1. Locate evidence bundle for target run.
2. Review manifest, checks, and benchmark summaries.
3. Confirm signatures, hashes, and coverage artifacts match expectations.

## Expected result

Evidence bundle supports a defensible operational or release decision.
