# Tutorial: Running Load Tests

## Goal

Run repeatable load scenarios and capture comparable metrics.

## Steps

1. Choose scenario and baseline dataset size.
2. Execute load command suite.
3. Compare latency and throughput against baseline reports.

## Expected result

Load reports expose performance profile and regression signals.
