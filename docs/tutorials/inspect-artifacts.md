# Tutorial: Inspect Artifacts

## Goal

Inspect generated artifacts, manifest structure, and evidence files.

## Steps

1. Locate generated artifact root.
2. Open manifest and validate digest references.
3. Confirm expected evidence files are present.

## Expected result

Artifact tree and manifest fields match documented release and ops expectations.
