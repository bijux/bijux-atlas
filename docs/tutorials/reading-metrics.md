# Tutorial: Reading Metrics

## Goal

Interpret key runtime, ingest, and query metrics for health verification.

## Steps

1. Open monitoring endpoint or dashboard datasource.
2. Inspect query latency, error rate, and ingest throughput metrics.
3. Compare current values to baseline thresholds.

## Expected result

Metric trends clearly indicate healthy or degraded service behavior.
