# Ops Local Full Quickstart

Use only these commands.

## 1. Prerequisites

```bash
make ops-prereqs
make ops-doctor
```

## 2. One-command local full proof

```bash
make ops-local-full
```

Expected output includes:
- `ops/_artifacts/<run_id>/report.json`

## 3. Useful follow-ups

```bash
make ops-artifacts-open
make ops-clean
```
