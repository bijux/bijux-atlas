# Ingest Performance Baseline

Baseline source: `tests/goldens/perf/ingest-benchmark-baseline.json`.

This baseline anchors throughput, latency, and memory expectations for small, medium, and large ingest workloads.

## Baseline Policy

- Baselines are updated only with fresh benchmark evidence.
- Baseline changes require explicit review and regression justification.
- Baseline drift without evidence is forbidden.
