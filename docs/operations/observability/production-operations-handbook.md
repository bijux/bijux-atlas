# Production Operations Handbook

- Owner: `bijux-atlas-operations`
- Type: `runbook`
- Audience: `operator`
- Stability: `stable`

## Operating Sequence

1. Start with runtime health and SLO dashboards.
2. Isolate affected domain through query, ingest, and registry dashboards.
3. Confirm impact with traces and logs.
4. Apply mitigation runbook steps.
5. Verify recovery with dashboard health summary and readiness report.
