# Dashboard Taxonomy

- Owner: `bijux-atlas-operations`
- Type: `reference`
- Audience: `operator`
- Stability: `stable`

## Categories

- Runtime: process health, uptime, resources, and liveness.
- Query: throughput, latency, cache efficiency, and errors.
- Ingest: ingest throughput, backlog, failures, and latency.
- Ops: registry access, drift visibility, and control-plane health.

## Usage

- Route incidents through category-specific dashboards before deep tracing.
- Keep category ownership aligned with on-call ownership.
