# Product quality ops bundle checklist

- Evidence collection runs with deterministic output.
- Evidence verifies offline.
- Secret redaction policy is enforced.
- Install, render, and validate evidence exists.
- Install matrix and render matrix reports exist.
- Schema, NetworkPolicy, and RBAC coverage reports exist.
- Inventory snapshot and tool versions are captured.
- Diff output is deterministic and flags high-risk paths.
