# Ops compatibility promise

- Chart version and workspace release version are aligned per release.
- Values schema changes follow compatibility review policy.
- Install matrix profile names remain stable across a release line unless explicitly deprecated.
- Breaking ops changes require migration notes and rollback guidance.
