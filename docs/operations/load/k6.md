# Atlas Load Harness (k6)

- Owner: `bijux-atlas-operations`

Deterministic load suites for performance gating.

## Run

```bash
$ ./scripts/perf/run_suite.sh mixed.json artifacts/perf/results
```

Expected output: k6 summary with request latency/error metrics.
