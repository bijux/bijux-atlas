# Performance Troubleshooting Guide

- Owner: `platform`
- Stability: `stable`
- Last verified against: `main@2228f79ef`

## Purpose

Provide a deterministic troubleshooting sequence for failing performance gates.

## Checklist

1. Verify baseline and candidate artifacts are parseable.
2. Re-run diff and regression detector scripts.
3. Inspect suite-specific threshold breaches.
4. Cross-check history and trend assets for drift.
5. Publish updated summary and audit artifacts.
