## Terms

- `baseline profile`: approved reference run used for comparison.
- `candidate profile`: current run being evaluated against baseline.
- `regression`: negative delta above policy threshold.
- `trend`: ordered history of key performance metrics.
- `anomaly`: out-of-pattern metric behavior requiring triage.
- `performance gate`: CI rule that blocks promotion on critical regressions.
