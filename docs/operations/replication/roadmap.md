# Replication Roadmap

## Near-Term Enhancements

- add automated promotion policies for health-triggered failover.
- add replica placement constraints per topology zone.
- add historical lag trend snapshots per shard.

## Mid-Term Enhancements

- add replica catch-up prioritization for hot shards.
- add quorum-aware read routing in query path.
- add rolling primary rotation tooling for maintenance.

## Long-Term Enhancements

- add multi-region replication controls.
- add durability tiering by dataset class.
- add policy-driven automated replica scaling.
