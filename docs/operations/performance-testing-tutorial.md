## Purpose

Provide a complete tutorial for running performance regression checks locally.

## Steps

1. Validate policy and generated assets with `jq`.
2. Run load suite coverage tests.
3. Run reproducibility and performance asset tests.
4. Generate a fresh regression report from baseline and candidate.
5. Review dashboard, trend, and summary artifacts.
