# Stack Components

The core stack components are declared under `ops/stack/` and exercised through
`make stack-up`, `make stack-down`, and related control-plane commands.

## Main Components

- runtime service
- Redis
- MinIO
- Prometheus
- Grafana
- OpenTelemetry collector
- Toxiproxy
