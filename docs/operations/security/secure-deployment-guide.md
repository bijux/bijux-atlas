# Secure Deployment Guide

Deployment requirements:

- TLS enabled with validated certificate material
- authentication and authorization controls enabled
- data protection policy loaded from governed config files
- integrity verification enabled for dataset artifacts

- `docs/operations/security/secure-deployment-guidelines.md`
