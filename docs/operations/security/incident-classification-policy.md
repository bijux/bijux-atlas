# Incident Classification Policy

Severity levels:

- critical: active compromise or broad integrity impact
- high: material security control bypass with limited blast radius
- medium: contained weakness requiring remediation
- low: minor issue with bounded impact

Status lifecycle:

- open -> contained -> mitigated -> resolved
