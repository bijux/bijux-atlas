# Threat Modeling Workshop Guide

Workshop agenda:

1. scope architecture boundaries
2. enumerate assets and sensitive data flows
3. classify threats with taxonomy
4. map mitigations and residual risk
5. run CLI verification and archive evidence

Recommended output:

- updated threat files
- updated registry and docs links
- fresh coverage and metrics artifacts
