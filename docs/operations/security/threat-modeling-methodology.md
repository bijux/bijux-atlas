# Threat Modeling Methodology

Atlas uses STRIDE-style threat classification with severity and likelihood scoring.

Workflow:

1. Identify assets and trust boundaries.
2. Map data flows and attack surfaces.
3. Enumerate threats by category.
4. Link mitigations and residual risk.
5. Validate threat registry coverage in CLI and CI.
