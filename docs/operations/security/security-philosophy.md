# Security Philosophy

Atlas security prioritizes deterministic evidence, explicit trust boundaries, and policy-as-code checks.

Principles:

- deny by default on privileged paths
- evidence-first validation for security claims
- reproducible controls over ad-hoc exceptions
- layered mitigation with residual risk documentation
