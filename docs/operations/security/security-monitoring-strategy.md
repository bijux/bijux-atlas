# Security Monitoring Strategy

Monitoring sources:

- audit events and threat verification reports
- integrity and tamper detection outputs
- vulnerability and dependency risk reports

Operational loop:

- detect: telemetry, logs, and report anomalies
- triage: classify by severity and impacted asset
- respond: execute security runbooks and evidence capture
